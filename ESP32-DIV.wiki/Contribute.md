# Contributing to ESP32-DIV

The **ESP32-DIV** is an open-source project, and we welcome contributions from the community! Whether you’re fixing bugs, adding features, improving documentation, or sharing ideas, your help makes the project better.

## How to Contribute
1. **Fork the Repository**: Fork the [ESP32-DIV GitHub repository](https://github.com/cifertech/ESP32-DIV).
2. **Clone and Work**: Clone your fork, make changes, and test them thoroughly.
3. **Submit a Pull Request**: Push your changes and create a pull request with a clear description.
4. **Follow Guidelines**: Ensure code follows the project’s style and includes documentation.

## Contact
Reach out via GitHub Issues or contact [CiferTech](https://cifertech.net/) for questions or collaboration ideas.
